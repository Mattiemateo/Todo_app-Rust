# Todo App
I created a Todo tracker CLI app in Rust. Just to learn the basic syntax and get to know the language. 

I have the following functions:
- todo_app add "text"
- todo_app list
- todo_app delete "Task-ID"
- todo_app done "Task-ID"
- todo_app purge
